# Scenario 01: Static secret leak

Мета сценарію: показати антипатерн, коли статичний секрет зберігається в CI/CD-пайплайні або конфігурації.

## Що демонструємо

1. Статичний пароль присутній у локальному `.env`-фрагменті.
2. Такий секрет легко витягується простим grep.
3. Це відповідає гілці дерева атаки для витоку через репозиторій/логи.

## Запуск

```bash
bash scenarios/01-static-secret-leak/reproduce.sh
```
